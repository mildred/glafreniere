Transparency.

Inside air, water or glass, the light waves constantly disturb electrons, which emit some new light. The original waves simply go through any object, but their action is cancelled. So the light seems to go through any transparent material, but one must realize that it is some new light.

A substance is transparent because it has a perfectly homogeneous structure. For instance, a crystal is made of regularly spaced atoms. All electrons inside it will react to any incident light wave and produce a new wave according to Huygens' Principle.

Because there are two sorts of electrons, which spin (phase) is opposite, just one half of them will be slightly pushed by the incident wave, making all of them emit on the same phase. All the wavelets will add themselves for rather long distances.
